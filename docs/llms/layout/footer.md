# Footer

> Page footer component.

## Import

```typescript
import { Footer } from '@iqbal-codes/ui-kit/blocks';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | **required** | Footer content |
| `className` | `string` | - | Additional CSS classes |

---

## Usage

```tsx
import { Footer } from '@iqbal-codes/ui-kit/blocks';

<Footer>
  <Text>© 2024 Company</Text>
</Footer>
```
